交通信息

三坊七巷及福州西湖公园

路线：福建医科大学上街校区 → 公交（如闽侯1路）至地铁2号线厚庭站 → 地铁2号线至东街口站（转地铁1号线至湖滨路站前往西湖公园） → 步行至三坊七巷（约5分钟）或西湖公园（约10分钟）。

平潭岛

路线：福建医科大学上街校区 → 公交至地铁2号线厚庭站 → 地铁2号线至福州汽车北站 → 长途汽车至平潭汽车站（票价约50元） → 平潭当地公交（如K1路）至龙凤头海滨浴场。

鼓山风景区

路线：福建医科大学上街校区 → 公交至地铁2号线厚庭站 → 地铁2号线至南门兜站 → 转公交（如7路、29路）至鼓山下院站 → 步行或缆车至景区入口。

福州国家森林公园

路线：从鼓山下院站 → 公交（如54路、72路）至森林公园站 → 步行至入口。

仓山老街

路线：福建医科大学上街校区 → 公交至地铁2号线厚庭站 → 地铁2号线至南门兜站 → 转地铁1号线至三叉街站 → 转公交（如124路）至仓山老街附近站 → 步行游览。

住宿与餐饮

住宿

地点：福建医科大学上街校区

地址：福州市闽侯县上街镇学园路1号

推荐餐厅

  • 福州鱼丸店：鱼丸汤，20-50元/人，地址：三坊七巷附近
  • 平潭海鲜餐厅：海鲜拼盘，100-200元/人，地址：龙凤头海滨浴场附近
  • 鼓山附近餐馆：福州炒面，30-60元/人
  • 校区附近餐馆：学生餐/快餐，10-30元/人，地址：福建医科大学上街校区周边
